120/*
121 * Official intel docs are somewhat sloppy concerning MI_LOAD_REGISTER_IMM:
122 * - Always issue a MI_NOOP _before_ the MI_LOAD_REGISTER_IMM - otherwise hw
123 * simply ignores the register load under certain conditions.
124 * - One can actually load arbitrary many arbitrary registers: Simply issue x
125 * address/value pairs. Don't overdue it, though, x <= 2^4 must hold!
